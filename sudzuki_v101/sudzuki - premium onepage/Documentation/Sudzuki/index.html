HTML Structure

Sudzuki is Bootstrap 3.1 based responsive theme. You can use all of bootstrap features. More information Bootstrap 3.1.

Customization

Sudzuki has easy customization features.

Color

Open style.css file and change this color #9cd05f

Flickr Widget

For account change, insert your ID after "user=YOUR ID". If you changed ID for other URL name, generate it on a service http://idgettr.com/

Example: '<script type="text/javascript" src="http://www.flickr.com/badge_code_v2.gne?count=6&display=latest&size=s&layout=x&source=user&user=34328769@N02">'

PHP Mail Form

Open assets/xml/config.php and change the mail address from line 6. Work only in FTP.
